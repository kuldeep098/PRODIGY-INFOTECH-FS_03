# 🛒 Local E-Commerce Store  

This project is part of my **3rd internship task at Prodigy Infotech**.  
The **Local E-Commerce Store** is a simple and responsive online shopping platform designed for local businesses to showcase and sell their products.  

---

## 🚀 Features  
- 📦 Product listing with clean and modern UI  
- 🛍️ Add to Cart functionality  
- 💳 Checkout system  
- 📱 Fully responsive design (works on mobile, tablet, and desktop)  
- ⚡ Focus on usability and performance  

---

## 🛠️ Tech Stack  
- **Frontend:** HTML, CSS, JavaScript  
- **Tools:** VS Code, Git, GitHub  

---

## 📂 Project Structure  
├── index.html # Home page with product listing
├── style.css # Styling for UI
├── script.js # Cart and checkout functionality
├── assets/ # Images and icons
└── README.md # Project documentation
